DOCUMENTATION DE L'API

INTRODUCTION

Cette documentation fournit des informations détaillées pour accéder à diverses ressources, mais aussi pour les modifier ou encore les supprimées.

ROUTES

/technologies

  • GET: Récupère la liste des technologies. Aucune donnée n'est nécessaire.

  • POST: Ajoute une nouvelle technologie. Données nécessaires (dans form-data):

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à lier' (ex : 1,8,10)

    • Key : 'logo' / Value : Chemin vers le fichier

  • PUT: Met à jour une technologie existante. Données nécessaires (dans le Header et dans Body > binary pour le logo) :

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'newname' / Value : 'Nouveau nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à lier' (ex : 1,8,10)

    • Selectionez le chemin vers le fichier du logo dans Body > binary

  • DELETE: Supprime des élements de la technologie, liés à la technologie et même la technologie dans son ensemble. Données nécessaires (dans x-www-form-encoded):

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à supprimer' (ex : 1,8,10)

    • Key : 'logo' (sans Value)

    • Key : 'all' (sans Value)
