DOCUMENTATION DE L'API

INTRODUCTION

Cette documentation fournit des informations détaillées pour accéder à diverses ressources, mais aussi pour les modifier ou encore les supprimées.

ROUTES

/technologies

  • GET: Récupère la liste des technologies. Aucune donnée n'est nécessaire.

  • POST: Ajoute une nouvelle technologie. Données nécessaires (dans form-data):

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à lier' (ex : 1,8,10)

    • Key : 'logo' / Value : Chemin vers le fichier

  • PUT: Met à jour une technologie existante. Données nécessaires (dans le Header et dans Body > binary pour le logo) :

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'newname' / Value : 'Nouveau nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à lier' (ex : 1,8,10)

    • Selectionez le chemin vers le fichier du logo dans Body > binary

  • DELETE: Supprime des élements de la technologie, liés à la technologie et même la technologie dans son ensemble. Données nécessaires (dans x-www-form-encoded):

    • Key : 'name' / Value : 'Nom de la technologie'

    • Key : 'categories' / Value : 'Identifiants des catégories à supprimer' (ex : 1,8,10)

    • Key : 'logo' (sans Value)

    • Key : 'all' (sans Value)

/categories

  • GET: Récupère la liste des categories. Aucune donnée n'est nécessaire.

  • POST: Ajoute une nouvelle catégorie. Donnée nécessaire (dans x-www-form-encoded):

    • Key : 'name' / Value : 'Nom de la nouvelle catégorie'

  • PUT: Met à jour une catégorie existante. Données nécessaires (dans x-www-form-encoded) :

    • Key : 'id' / Value : 'Identifiant de la catégorieque vous souhaitez modifier'

    • Key : 'name' / Value : 'Nouveau nom de la catégories'

  • DELETE: Supprime une catégorie. Donnée nécessaire (dans x-www-form-encoded):

    • Key : 'name' / Value : 'Nom de la catégorie'

/ressources

  • GET: Récupère la liste des ressources. Aucune donnée n'est nécessaire.

  • POST: Ajoute une nouvelle ressource. Donnée nécessaire (dans x-www-form-encoded):

    • Key : 'id' / Value : 'Identifiant de la technologie concernée'

    • Key : 'url' / Value : 'URL de la ressource'

  • PUT: Met à jour une ressource existante. Données nécessaires (dans x-www-form-encoded) :

    • Key : 'id' / Value : 'Identifiant de la technologie concernée'

    • Key : 'url' / Value : 'Nouvel URL de la ressource'

  • DELETE: Supprime une ressource. Donnée nécessaire (dans x-www-form-encoded):

    • Key : 'id' / Value : 'Identifiant de la ressource concernée'
